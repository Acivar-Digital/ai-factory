Build a python script named dm_strength.py in the temp folder
(admin/orchestrator/temp/) that, given a birth date/time and sex, prints
the Bazi Four Pillars (year/month/day/hour stems and branches), identifies
the Day Master (DM), and prints a simple Day Master strength estimate
(Strong / Neutral / Weak) with a one-line reason.

Requirements:
- Write the file via the write_file tool to admin/orchestrator/temp/dm_strength.py
- It must run with `python admin/orchestrator/temp/dm_strength.py` and print output
- Keep it dependency-light and runnable (you may use the lunar-python
  library which is already installed in this environment)
- No comments needed; just working code
